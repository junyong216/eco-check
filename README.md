## 🛠 사용 기술 (Tech Stack)
- **Framework:** Next.js 14 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Deployment:** Vercel
- **SEO/Analytics:** Google Search Console, Google AdSense

## ✨ 핵심 기능 및 구현 상세
### 1. 스마트 용어 사전 (가나다순 정렬)
- `localeCompare('ko')`를 활용하여 한국어 사전식 정렬 로직 구현
- 사용자 편의를 위한 실시간 필터링 검색 시스템

### 2. 하이브리드 테마 시스템
- 다크모드/라이트모드 완벽 대응 및 UI 일관성 유지
- CSS 변수를 활용한 브랜드 컬러(Red) 포인트 디자인

### 3. 검색 엔진 최적화 (SEO)
- Metadata API를 이용한 페이지별 최적화 및 구글 소유권 인증
- 사이트맵(Sitemap) 및 robots.txt 자동 생성 적용
